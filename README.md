# TEE-for-Cloud-Computing
Trusted Execution Environments (TEEs) for Secure Cloud Computing

This repository contains my project on Trusted Execution Environments (TEEs) and how they can make cloud computing more secure. The work compares technologies such as Intel SGX, Intel TDX, and AMD SEV-SNP to understand the balance between security and system performance.

To study this, I built a Python + Streamlit simulation dashboard that visualizes metrics like latency, throughput, memory overhead, and scalability using graphs and tables. The repository also includes my project report, IEEE research paper, and presentation files.

Main Findings
1.Intel SGX offers the strongest security but with higher overhead
2.Intel TDX provides a balanced mix of security and performance
3.AMD SEV-SNP gives the best performance and scalability
